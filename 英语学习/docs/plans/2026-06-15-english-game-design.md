# English Fun Zone — 英语学习游戏设计文档

> 日期：2026-06-15
> 状态：✅ 已确认

## 项目定位

Web 端英语学习休闲游戏合集，自适应难度（A1-C2），覆盖单词、语法、听力、发音四大技能。

## 目标用户

通用/不限年龄，自适应难度调节。

## 平台

Web 网页版，浏览器打开即玩。

## 技术选型

| 层 | 技术 |
|----|------|
| 前端框架 | React + TypeScript |
| 样式 | Tailwind CSS |
| 状态管理 | Zustand |
| 语音 | Web Speech API |
| 数据存储 | Supabase |

## MVP 游戏模块

| 游戏 | 技能 | 玩法 |
|------|------|------|
| 🧩 拼词大作战 | 单词拼写 | 限时内用乱序字母拼出正确单词 |
| 🔗 单词连连看 | 词汇量 | 英文-中文配对消除 |
| ✏️ 语法改错 | 语法 | 展示错误句子，点击修正 |
| 🎧 听音选词 | 听力 | 播放发音，4选1 |

## 数据库表

| 表名 | 用途 |
|------|------|
| user_profiles | 用户档案 |
| game_scores | 游戏记录 |
| achievements | 成就徽章 |
| daily_challenges | 每日任务 |
| word_bank | 分级词库 |

## 路由

| 路径 | 页面 |
|------|------|
| / | 主页大厅 |
| /game/spell | 拼词大作战 |
| /game/match | 单词连连看 |
| /game/grammar | 语法改错 |
| /game/listen | 听音选词 |
| /profile | 个人中心 |
| /placement | 初始定级测试 |

## 自适应难度

- 定级测试 20 题定级
- 正确率 ≥ 80% → 难度 +1
- 正确率 ≤ 40% → 难度 -1

## 趣味机制

- 每日挑战 + 连击加分
- 成就徽章系统
- 每周学习报告
