## 四、技术亮点

1. **无框架依赖** - 纯原生 JavaScript ES Module，零构建配置
2. **经典扫雷体验** - CSS 3D 凸起/凹陷效果，数字颜色方案
3. **程序化音效** - 使用 Web Audio API 生成音效，无需外部音频文件
4. **安全首点** - Fisher-Yates 洗牌算法，首次点击及周围 8 格 100% 安全
5. **防作弊** - 后端时间合理性校验、参数完整性校验、频率限制
6. **响应式设计** - 3 个 CSS 断点，移动端触屏支持（长按标记）
7. **安全架构** - helmet + CORS + rate-limit + RLS + 前后端双重校验
8. **完整测试** - 143 个测试用例，核心模块 > 95% 覆盖率

## 五、启动方式

### 1. 配置环境

```bash
cd e:\AI编程\扫雷\server
cp .env.example .env
# 编辑 .env 填入 Supabase 配置
```

### 2. 初始化数据库

在 Supabase SQL Editor 中执行 `server/init-db.sql`

### 3. 安装依赖

```bash
cd e:\AI编程\扫雷\server
npm install
```

### 4. 启动服务

```bash
npm start
# 访问 http://localhost:3000
```

### 5. 运行测试

```bash
npm test              # Jest 测试
node tests/game.test.js    # 游戏逻辑测试
node tests/api.test.js     # API 测试（需服务器运行）
```
